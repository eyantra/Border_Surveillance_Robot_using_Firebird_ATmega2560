00006 """
00007 highgui_win32
00008 
00009 UI background window thread implementation for Windows, since OpenCV doesn't
00010 support it natively.  Wraps (and is safe for import * from) the following
00011 OpenCV functions:
00012 
00013     cvStartWindowThread
00014     cvNamedWindow
00015     cvDestroyWindow
00016     cvDestroyAllWindows
00017     cvShowImage
00018     cvSetMouseCallback
00019     cvCreateTrackbar
00020     cvWaitKey
00021 
00022 If cvStartWindowThread is not called, the other functions are just pass
00023 through to their normal wrappers in the opencv.highgui module.  However,
00024 once cvStartWindowThread is called, those functions execute calls to the
00025 equivalent opencv.highgui methods from within a background thread, and that
00026 thread maintains an event loop, calling cvWaitKey periodically.
00027 
00028 By default, calls are synchronized with the window thread, so for example
00029 a foreground call to cvShowImage won't return until the window thread has
00030 called the actual cvShowImage method.
00031 
00032 There is latency added by the synchronization due to the round trip thread
00033 context switching.  Callers requiring the least impact to the foreground
00034 thread can set the optional cvStartWindowThread 'synchronized' parameter to
00035 False, but should appreciate that access to function parameters by the
00036 window thread may be delayed.  For example, the foreground should not
00037 immediately manipulate an image used as a parameter to cvShowImage.  Memory
00038 usage may also be slightly higher if many commands by the foreground thread
00039 are enqueued before being executed by the window thread.
00040 """

00093     def run(self):
00094         while 1:
00095             if self.stopping:
00096                 hg.cvDestroyAllWindows()
00097                 return
00098 
00099             try:
00100 
00101                 # Check for a new command request
00102                 try:
00103                     (self.func,
00104                      self.args,
00105                      self.kwargs) = self.cmds.get_nowait()
00106                 except Empty:
00107                     self.func = None
00108 
00109                 # Execute function if requested
00110                 if self.func:
00111                     try:
00112                         rc = self.func(*self.args, **self.kwargs)
00113                     except Exception:
00114                         rc = None
00115                         sys.stderr.write('ImageThread exception - ignoring:')
00116                         traceback.print_exc()

00118                         self.result.put(rc)

00122                 # a small number in case WaitKey isn't called by the thread.

00127                     rc = hg.cvWaitKey(5)
00128                     if rc >= 0:

00133                     # Yield CPU to prevent CPU burn
